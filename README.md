# YYN Wiki 项目介绍

YYN Wiki 是一个面向算法学习与知识整理的轻量级静态知识库项目。项目整体风格参考 OI Wiki，主要用于沉淀 Python 算法模板、算法思想讲解、复杂度分析、图示说明以及常见易错点，方便个人复习、查阅和长期维护。

## 项目定位

YYN Wiki 的目标不是只保存代码模板，而是将算法内容整理成更适合阅读和学习的知识条目。每个算法页面会尽量包含算法背景、核心思想、状态定义或关键性质、公式推导、代码实现、复杂度分析、常见问题和图示辅助理解。

项目适合用于个人算法学习笔记整理、Python 算法模板归档、算法竞赛复习、常用算法和数据结构的快速查阅，以及将零散笔记逐步整理成体系化知识库。

## 内容结构

YYN Wiki 按算法知识大类进行组织，顶部导航栏展示主要分类，进入某个大类后，左侧导航栏展示该分类下的具体算法主题。

目前知识库主要包含基础算法、前缀与差分、数据结构、图论与树、动态规划、数学、字符串等方向。

## 页面风格

YYN Wiki 的页面会尽量采用“讲解 + 模板”的形式，而不是单纯堆放代码。一个较完整的算法页面通常包含算法介绍、核心思想、公式或状态定义、算法流程、Python 模板代码、复杂度分析、常见易错点和图示辅助理解。

## 图示与交互

为了让算法过程更直观，YYN Wiki 会在部分页面中加入图示说明。对于多步过程，例如 LIS 的数组更新、背包 DP 的枚举方向、Dijkstra 的松弛过程等，项目支持使用手动轮播图进行展示。

这种方式相比单张静态图更适合解释算法执行过程，读者可以按步骤查看每一轮状态变化，从而更容易理解算法细节。

## 技术特点

YYN Wiki 是一个静态知识库项目，适合托管在 GitHub Pages 上。项目内容主要使用 Markdown 编写，便于长期维护、版本管理和持续更新。

项目支持中文页面、中文搜索、数学公式、代码高亮、图示说明、手动轮播图、顶部导航和侧边栏导航，也适合个人长期维护和扩展。

## 项目目标

YYN Wiki 希望逐步形成一个结构清晰、内容完整、便于查阅的 Python 算法知识库。它既可以作为日常刷题时的模板速查站，也可以作为复习算法思想和整理竞赛知识体系的长期笔记站。

项目的长期目标是将常用算法模板体系化整理，为每个算法补充清晰解释，用图示降低理解难度，并保持页面结构统一，让知识库既适合快速查模板，也适合系统学习。
