# 字符串总览

字符串算法常见目标是：在大量字符中快速完成匹配、比较、查询前缀关系、判断子串相等、处理异或前缀等操作。

本章主要包含：

- **KMP**：利用模式串自身的前后缀信息，在失配时避免主串指针回退。
- **Z 函数**：计算每个后缀和原串的最长公共前缀，常用于模式匹配。
- **字符串哈希**：把字符串映射成数字，用前缀哈希快速比较子串。
- **Trie**：把字符串集合组织成树，适合前缀查询和单词匹配。
- **01 Trie**：把整数二进制位组织成 Trie，常用于最大异或。

!!! note "学习方式"
    字符串算法不要只背模板。更重要的是弄清楚：模板里的每个指针为什么移动、为什么失配时能跳过一部分比较、为什么前缀数组或哈希数组能支持快速查询。
