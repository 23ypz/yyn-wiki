# 基础算法总览

基础算法部分用于整理最常用、最容易在题目中反复出现的算法技巧。本章不是单纯保存代码，而是按照“问题模型 → 核心不变量 → 模板代码 → 复杂度 → 易错点”的方式组织内容。

本章当前包含：

- **二分**：包括基础二分和二分答案。
- **排序**：包括简单排序和自定义排序。
- **位运算**：当前重点整理线性基。

这些内容看起来基础，但实际比赛或刷题中非常容易因为边界、单调性、比较规则写错。因此本章会尽量强调每个模板背后的“不变量”和“适用条件”。

## 学习建议

二分要先判断自己找的是“第一个满足条件的位置”还是“最后一个满足条件的位置”；排序要先明确排序关键字，而不是边写边猜；线性基要理解它维护的是异或意义下的线性空间，而不是普通数组集合。
