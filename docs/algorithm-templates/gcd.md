# 最大公约数

## 定义

两个整数 $a,b$ 的最大公约数记作：

$$
\gcd(a,b)
$$

它表示能够同时整除 $a$ 和 $b$ 的最大正整数。

## 欧几里得算法

欧几里得算法基于下面的性质：

$$
\gcd(a,b)=\gcd(b,a\bmod b)
$$

当 $b=0$ 时：

$$
\gcd(a,0)=a
$$

因此可以不断将 $(a,b)$ 替换为 $(b,a\bmod b)$，直到第二个数变为 $0$。

## 例题

!!! example "例题：最大公约数"
    给定两个整数 $a,b$，求它们的最大公约数。

    例如 $a=24,b=36$，答案为 $12$。

## 解释

以 $\gcd(36,24)$ 为例：

$$
\gcd(36,24)=\gcd(24,12)=\gcd(12,0)=12
$$

每一步都不会改变最大公约数，但会让数值变小。

## 实现

```python
def gcd(a, b):
    """欧几里得算法求最大公约数"""
    while b:
        a, b = b, a % b
    return a

print(gcd(24, 36))
```

## 复杂度

欧几里得算法的时间复杂度为：

$$
O(\log \min(a,b))
$$

空间复杂度为 $O(1)$。

## 易错点

- 如果输入可能有负数，通常先取绝对值：`gcd(abs(a), abs(b))`。
- Python 标准库中也有 `math.gcd`，比赛中可以直接使用。
- 多个数的最大公约数可以从左到右依次合并。
