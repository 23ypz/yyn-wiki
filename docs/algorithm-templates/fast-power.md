# 快速幂

## 定义

快速幂用于在对数时间内计算：

$$
a^b \bmod p
$$

如果直接连乘 $b$ 次，时间复杂度是 $O(b)$。当 $b$ 很大时，例如 $10^{18}$，直接乘法无法接受。快速幂的核心是利用指数的二进制表示，把指数拆成若干个 $2$ 的幂。

## 性质

设：

$$
b = \varepsilon_0 2^0 + \varepsilon_1 2^1 + \varepsilon_2 2^2 + \cdots
$$

其中 $\varepsilon_i \in \{0,1\}$，则：

$$
a^b = \prod_{\varepsilon_i = 1} a^{2^i}
$$

每次循环把当前底数平方：

$$
a,\ a^2,\ a^4,\ a^8,\ldots
$$

当指数二进制当前位为 $1$ 时，将当前底数乘入答案。

## 例题

!!! example "例题：求模幂"
    给定整数 $a,b,p$，求 $a^b \bmod p$。

    例如 $a=2,b=10,p=10^9+7$，答案为 $1024$。

## 解释

以 $2^{10}$ 为例：

$$
10 = (1010)_2 = 2^3 + 2^1
$$

所以：

$$
2^{10} = 2^8 \times 2^2
$$

快速幂只需要不断平方底数，并根据二进制位决定是否乘入答案。

## 实现

```python
def ksm(a, b, p):
    """快速幂计算 a^b % p"""
    res = 1
    while b:
        if b & 1:
            res = (res * a) % p
        a = (a * a) % p
        b >>= 1
    return res

print(ksm(2, 10, 1000000007))
```

## 复杂度

指数 $b$ 每次右移一位，所以循环次数约为 $\log_2 b$。

- 时间复杂度：$O(\log b)$
- 空间复杂度：$O(1)$

## 易错点

- 如果需要取模，每次乘法后都应该 `% p`，避免数值过大。
- 当 $b=0$ 时，按照约定 $a^0=1$，代码会直接返回 `1`。
- 如果题目需要普通幂而不是模幂，可以去掉 `% p`，但数值可能非常大。
