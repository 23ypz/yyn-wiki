# 质数筛

## 定义

质数筛用于批量找出 $[1,N]$ 范围内的所有质数。最常用的是埃氏筛。

## 埃氏筛思想

如果 $i$ 是质数，那么 $i$ 的倍数一定不是质数。于是从 $i^2$ 开始，将：

$$
i^2,\ i^2+i,\ i^2+2i,\ldots
$$

全部标记为合数。

为什么可以从 $i^2$ 开始？因为小于 $i^2$ 的倍数都已经被更小的质因子筛掉了。

## 例题

!!! example "例题：输出前若干个质数"
    给定上限 $N$，求不超过 $N$ 的所有质数。

## 实现

```python
mx = 10 ** 6 + 1
isp = [True] * mx
primes = []

for i in range(2, mx):
    if isp[i]:
        primes.append(i)
        for j in range(i * i, mx, i):
            isp[j] = False

isp[0] = isp[1] = False

print(primes[:10])
```

## 复杂度

埃氏筛的时间复杂度通常记为：

$$
O(N\log\log N)
$$

空间复杂度为：

$$
O(N)
$$

## 易错点

- `0` 和 `1` 不是质数。
- 标记倍数时从 `i * i` 开始，而不是从 `2 * i` 开始，可以减少重复标记。
- 如果 `mx` 表示数组长度，那么筛的范围是 $[0,mx-1]$。
