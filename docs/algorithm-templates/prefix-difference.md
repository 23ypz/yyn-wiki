# 前缀和与差分总览

前缀和与差分是一组互为逆操作的基础技巧。

- **前缀和**适合处理大量区间查询。它先预处理累计信息，再把一次区间查询降为常数次计算。
- **差分**适合处理大量区间修改。它不直接修改区间内每一个元素，而是在边界位置打标记，最后统一还原。

它们的关系可以简单理解为：

\[
a \xrightarrow{做差} diff,\qquad diff \xrightarrow{前缀和} a
\]

如果题目是“数组不怎么改，但经常查询区间和”，优先考虑前缀和；如果题目是“经常给一段区间加值，最后才问结果”，优先考虑差分。

## 本章结构

本章拆成四个独立小节：

- 一维前缀和：处理数组区间和。
- 二维前缀和：处理矩阵子矩形和。
- 一维差分：处理数组区间加。
- 二维差分：处理矩阵子矩形加。

!!! tip "核心区别"
    前缀和是“先累加，后相减”；差分是“先打标记，后累加”。
