# 数学总览

数学部分整理的是算法竞赛中最常用的一批数论与组合计数工具。它们通常不是单独出现，而是作为图论、动态规划、字符串哈希、组合计数等题目的基础模块。

本节按照 OI Wiki 式的结构，将原本集中在“数论基础”中的模板拆成多个独立页面，方便从左侧导航快速定位。

## 本章内容

- [快速幂](fast-power.md)：用二进制拆分指数，快速计算幂和模幂。
- [最大公约数](gcd.md)：用欧几里得算法求两个整数的最大公约数。
- [最小公倍数](lcm.md)：结合最大公约数计算最小公倍数。
- [矩阵快速幂](matrix-fast-power.md)：把线性递推转化为矩阵乘法并快速求高次幂。
- [组合数](combination.md)：预处理阶乘和逆阶乘，快速求组合数。
- [质因数分解](prime-factorization.md)：将整数分解成若干质数幂的乘积。
- [质数筛](prime-sieve.md)：批量预处理一定范围内的质数。
- [逆元](modular-inverse.md)：在模意义下处理除法。

## 常见符号

设模数为 $p$，常见写法为：

$$
a \bmod p
$$

表示 $a$ 除以 $p$ 的余数。在代码中一般写成：

```python
a % p
```

当结果需要对大质数取模时，常用：

$$
\mathrm{mod} = 10^9 + 7
$$

## 学习顺序

建议按下面顺序学习：

1. 先学 [最大公约数](gcd.md)、[最小公倍数](lcm.md)。
2. 再学 [快速幂](fast-power.md) 和 [逆元](modular-inverse.md)。
3. 然后学 [组合数](combination.md)。
4. 最后学习 [质数筛](prime-sieve.md)、[质因数分解](prime-factorization.md)、[矩阵快速幂](matrix-fast-power.md)。

!!! tip "使用建议"
    数学模板最容易出错的地方通常不是公式本身，而是边界条件、模数是否为质数、数组预处理上限是否足够、以及整数下标是否从 0 或 1 开始。
