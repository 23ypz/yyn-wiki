# 欢迎来到 YYN Wiki

YYN Wiki 是一个面向算法学习、竞赛训练和 Python 模板整理的个人知识库。它的目标不是只保存几段代码，而是把常用算法整理成可以长期查阅的讲解型页面：先说明问题模型和核心思想，再给出公式、不变量、代码模板、复杂度分析、易错点和图示过程。

!!! note "项目定位"
    这里更像一本持续更新的算法手册，而不是零散笔记。你可以把它当成刷题时的模板库，也可以把它当成复习算法知识体系的目录。

## 这个站点包含什么

- **比赛相关**：介绍 OI、ICPC/CCPC、Codeforces、AtCoder、LeetCode、Kaggle 等常见竞赛与训练平台。
- **基础算法**：二分、排序、位运算与线性基等常用技巧。
- **前缀与差分**：一维/二维前缀和，一维/二维差分。
- **数据结构**：并查集、堆、单调栈、单调队列、SortedList、树状数组、线段树、ST 表等。
- **图论与树**：BFS、DFS、最短路、LCA、树上差分等。
- **动态规划**：简单 DP、背包 DP、区间 DP、树形 DP、数位 DP。
- **数学与字符串**：数论基础、组合数、逆元、KMP、Z 函数、字符串哈希、Trie 等。

## 页面怎么阅读

每个算法页面通常按下面的顺序组织：

1. 先看**经典例题**，明确这个算法解决什么问题。
2. 再看**核心思想和公式**，理解状态、边界、不变量或转移来源。
3. 然后看**模板代码**，把代码和前面的思想一行一行对应起来。
4. 最后看**复杂度和易错点**，避免在实际写题时踩坑。

部分页面还加入了手动轮播图。遇到 BFS、Dijkstra、LIS、线段树、树状数组这类“过程型算法”时，可以按步骤查看每一轮状态变化。

## 推荐入口

- 想了解常见比赛：从 [比赛相关](contest/index.md) 开始。
- 想查算法目录：进入 [算法知识总览](algorithm-templates/index.md)。
- 想快速复习 DP：进入 [动态规划总览](algorithm-templates/dynamic-programming.md)。
- 想查模板：直接在右上角搜索算法名，例如 `Dijkstra`、`线段树`、`KMP`。

!!! tip "维护建议"
    新增内容时，建议保持“例题 → 原理 → 模板 → 复杂度 → 易错点”的结构。这样以后复习时不只是在复制模板，而是能快速回忆模板为什么这样写。
