# 比赛相关总览

本栏目整理大学生阶段常见的算法竞赛。这里的“算法竞赛”主要指以程序设计、算法设计、数据结构、数学建模式思维和限时调试能力为核心的竞赛，而不是软件工程项目赛或论文创新赛。

!!! warning "信息时效"
    竞赛赛制、报名时间、晋级方式和获奖比例可能每年调整。本栏目用于帮助你理解比赛定位和备赛方向，正式报名和评奖规则一定要以当年官网、学校教务处通知或组委会章程为准。

## 适合大学生关注的竞赛

### ICPC

ICPC 是国际大学生程序设计竞赛，通常被认为是大学生算法竞赛中最核心、影响力最大的体系之一。它强调三人组队、五小时、多题目、罚时排名和团队协作。

适合目标：想系统训练算法能力、代表学校参赛、追求高水平程序设计竞赛成绩的同学。

### CCPC

CCPC 是中国大学生程序设计竞赛，赛制与 ICPC 风格接近，更贴近国内高校竞赛体系。它通常也是学校程序设计队伍重点备赛的赛事。

适合目标：希望参加国内高水平大学生程序设计竞赛、积累区域赛/邀请赛经验的同学。

### XCPC

XCPC 不是某一个单独比赛的官方名称，而是国内竞赛圈常用的泛称，通常指 ICPC、CCPC 以及相同赛制的校赛、邀请赛、区域赛、训练赛等一整类“大学生程序设计竞赛”。

适合目标：想系统理解“ACM 赛制”训练体系，不只盯着单个比赛的同学。

### 蓝桥杯

蓝桥杯是国内参与人数非常多的大学生软件与信息技术类竞赛。程序设计方向一般是个人赛，题型覆盖基础语法、算法、数据结构和综合应用。

适合目标：低年级入门、想获得省级/国家级竞赛经历、需要以个人身份参赛的同学。

### 码蹄杯

码蹄杯全国大学生程序设计大赛近年在高校中逐渐常见，采用在线评测和类 ACM 赛制，覆盖算法、数据结构、搜索、动态规划、字符串、图论、计算几何等内容。

适合目标：有一定刷题基础，希望通过个人赛或分阶段赛事检验算法能力的同学。

### 团体程序设计天梯赛

天梯赛强调团队总成绩，由多名队员共同组成学校队伍参加。它既考个人做题能力，也考学校整体梯队建设。

适合目标：学校统一组织参赛、希望在团队环境中承担稳定得分任务的同学。

### 百度之星

百度之星是面向算法设计和程序设计能力的竞赛，初赛通常为线上赛，决赛更强调较强的算法综合能力和排名竞争。

适合目标：有一定算法基础，想挑战更高难度个人赛、接触企业背景算法赛事的同学。

## 如何选择优先级

如果你刚入门，可以先从蓝桥杯、校内赛、周赛和简单 OJ 训练开始；如果已经掌握基础数据结构和常见算法，可以参加天梯赛、码蹄杯和一些校内 XCPC 训练赛；如果目标是高水平算法竞赛，则应长期准备 ICPC / CCPC，并稳定参加 Codeforces、AtCoder、牛客等线上训练。

粗略路线可以理解为：

```text
基础语法与简单算法
  ↓
蓝桥杯 / 校赛 / 周赛
  ↓
天梯赛 / 码蹄杯 / XCPC 训练赛
  ↓
CCPC / ICPC 区域赛
  ↓
更高层次区域赛奖项、总决赛或世界总决赛目标
```
