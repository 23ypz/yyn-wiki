# 蓝桥杯

蓝桥杯全国大学生软件和信息技术大赛是国内参与人数较多的大学生竞赛之一。对于算法方向学生来说，最常见的是软件类程序设计相关赛项。

## 竞赛定位

蓝桥杯更适合算法竞赛入门到中级阶段的同学。它的优势是参赛门槛相对低、覆盖学校多、个人赛形式清晰、获奖层级明确，适合低年级同学积累竞赛经验。

相比 ICPC / CCPC，蓝桥杯更偏个人能力考察；相比纯在线周赛，它又具有正式赛事、证书和学校认可度。

## 参赛对象

蓝桥杯面向高校学生。不同赛项可能按语言、专业方向、学历层次或组别划分，例如 C/C++、Java、Python、软件测试、电子类等。算法学习者通常重点关注程序设计相关组别。

## 赛制

蓝桥杯常见形式是个人赛，参赛者在规定时间内完成若干道题。题型一般包括：

- 填空题或结果填空题。
- 编程题。
- 模拟、枚举、搜索、动态规划、图论、数论等算法题。

近年不同赛项的具体考试形式会随组别调整，因此报名时要看当年赛项说明。

## 获奖率

蓝桥杯一般分为省赛和全国总决赛。省赛常见设置一、二、三等奖，全国总决赛也会设置不同等级奖项。近年来不同届别、组别的比例可能不同，例如有通知中提到省赛奖项按一定比例设置，一等奖、二等奖、三等奖分别对应若干比例；总决赛也会根据成绩设置一、二、三等奖和优秀奖。

!!! warning "比例不是固定分数线"
    蓝桥杯的获奖比例通常只是参考。组委会可能根据题目难度、整体答题情况和最低分数线调整实际获奖情况，零分或未达最低线通常不能获奖。

## 难度评价

蓝桥杯难度通常可以分成三段：

### 省赛入门题

这部分适合新手，常见内容包括语法、模拟、日期处理、枚举、排序、简单数学等。认真刷基础题后比较容易拿分。

### 省赛中档题

中档题开始考察算法能力，例如二分、前缀和、DFS/BFS、动态规划、并查集、最短路、贪心等。想拿较高奖项，通常需要解决这部分题。

### 国赛题

国赛难度明显提高，题目更综合，代码细节更多。想在国赛中拿好奖，需要接近 XCPC 中档训练水平，而不是只会基础语法。

## 和 ICPC/CCPC 的区别

| 项目 | 蓝桥杯 | ICPC/CCPC |
| --- | --- | --- |
| 参赛形式 | 多为个人赛 | 通常三人组队 |
| 训练重点 | 基础算法和个人实现 | 综合算法、团队策略、复杂题 |
| 难度跨度 | 入门到中高 | 中高到顶尖 |
| 适合阶段 | 大一大二入门、积累证书 | 长期校队训练 |

## 备赛建议

1. 先刷历年真题，熟悉题型和输入输出风格。
2. 不要只背代码，要理解数据范围对应的复杂度。
3. 重点掌握：枚举、前缀和、二分、DFS/BFS、DP、并查集、最短路、数论基础。
4. 练习时要限时完成，因为蓝桥杯很考验个人速度和稳定性。
5. 省赛前以基础题和中档题为主，国赛前再加强图论、DP 和综合题。

## 适合谁参加

蓝桥杯适合大多数算法学习者，尤其适合：

- 刚开始参加大学生竞赛的同学。
- 想用个人赛检验算法基础的同学。
- 需要正式竞赛证书和项目经历的同学。
- 准备进一步参加 ICPC / CCPC 的低年级同学。
