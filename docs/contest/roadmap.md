# 大学生算法竞赛路线建议

不同比赛的定位不同，不建议一上来就把所有比赛都报满。更合理的做法是根据自己的阶段选择合适目标。

## 阶段一：刚入门

适合目标：蓝桥杯、校内新生赛、平台周赛、天梯赛 L1。

重点能力：

- 熟练输入输出。
- 掌握数组、字符串、函数、排序。
- 会写模拟、枚举、简单数学。
- 了解时间复杂度。

推荐训练：

```text
语法基础 → 模拟题 → 排序 → 枚举 → 前缀和 → 简单搜索
```

## 阶段二：有基础但不稳定

适合目标：蓝桥杯省赛、码蹄杯初赛、天梯赛 L2、校赛选拔。

重点能力：

- 二分、前缀和、差分。
- DFS/BFS。
- 贪心和简单 DP。
- 并查集、堆、基础图论。
- 基础数论和组合计数。

推荐训练：

```text
专题学习 → 模板整理 → 限时做题 → 赛后补题
```

## 阶段三：准备 XCPC

适合目标：CCPC、ICPC、XCPC 邀请赛、牛客多校、校队训练赛。

重点能力：

- 系统图论：最短路、生成树、连通性、网络流入门。
- 系统 DP：背包、区间、树形、状压、数位、概率。
- 数据结构：线段树、树状数组、ST 表、Trie、哈希。
- 数学：组合、逆元、筛法、矩阵快速幂、容斥。
- 团队协作：分工、读题、罚时策略、代码风格统一。

推荐训练：

```text
每周组队赛 → 补题到会 → 专题突破 → 整理模板 → 模拟正式赛
```

## 阶段四：冲击高水平奖项

适合目标：ICPC / CCPC 区域赛银牌、金牌，总决赛或更高层次目标。

重点能力：

- 中高难题建模。
- 多算法融合。
- 高质量代码实现。
- 快速构造反例和随机测试。
- 队伍配合稳定。

这个阶段的提升通常来自大量高质量训练，而不是单纯刷题数量。每场训练后的复盘、补题和总结非常关键。

## 比赛选择建议

如果你是大一：

```text
蓝桥杯 + 校赛 + 天梯赛 + 平台周赛
```

如果你是大二：

```text
蓝桥杯国赛 / 码蹄杯 / 天梯赛 / XCPC 训练赛 / CCPC 网络赛
```

如果你已经进入校队：

```text
ICPC + CCPC + 多校训练 + 邀请赛 + 高质量线上赛
```

如果你主要想就业面试：

```text
蓝桥杯 / LeetCode / 基础算法训练 / 部分企业算法赛
```

## 不同比赛的粗略定位

| 比赛 | 推荐阶段 | 难度 | 主要价值 |
| --- | --- | --- | --- |
| 蓝桥杯 | 入门到中级 | 中等 | 个人赛、证书、基础算法检验 |
| 码蹄杯 | 中级 | 中等偏上 | 类 ACM 赛制、阶段性检验 |
| 天梯赛 | 入门到进阶 | 分层明显 | 团队总分、学校组织训练 |
| 百度之星 | 中高级 | 较高 | 个人算法能力挑战 |
| CCPC | 进阶到高水平 | 高 | 国内大学生程序设计主线赛事 |
| ICPC | 高水平 | 高到顶尖 | 国际大学生程序设计主线赛事 |
| XCPC | 训练体系 | 浮动大 | 覆盖校赛、邀请赛、多校、区域赛 |

## 最重要的建议

比赛只是训练结果的检验。真正提升算法能力的关键是：

1. 有计划地学专题。
2. 限时做题。
3. 赛后补题。
4. 整理模板。
5. 复盘错误。
6. 持续参加训练。

如果你把每场比赛都当作一次“发现短板”的机会，长期进步会比只关注获奖更稳定。
